Standard Stuff

These are the most important and most used features in Doctasm.

Easy Editing

We avoid HTML (complex) and WYSIWYG (sounds good but is very bad for responsive layout). Instead we use the Doctasmic plain text format.

Style Templates

We provide several great style templates but you can add your own with CSS control.

Table of Contents

We have made this really great and a no brainer. All drag-and-drop with customizable icons. A pleasure.

Instant Preview

With the click of a button you can see your docs exactly as they will appear for real.

Search

All page titles are ready to search and you can add tags to each page for targeted searching.

Doc Asset Management

Doctasm lets you upload any images and files that can be shown in or linked from your content.
TOP ▲

Formatting Options

Your formatting options are virtually limitless. Our special plain text doc language **Doctasmic** comes with plenty of document stylings but you can add anything you like with a little tinkering.

Simple Font Control

All body and header fonts can be controlled via the easy to use Doctasm menus.

Body Text

This is the main body text.

Headers

You can have as many different header styles as you like although one is normally right for clean docs.

Lists

We support bullets, numbered and custom list styles.

Code

If you are a software company and want to include code examples including special formatting for standard languages.

Special Callout Styles

We support a range of styles such as Important, Tips & Tricks, Examples as well as being able to add user definable ones.

Images & Videos

You can embed images (with click and zoom support) and videos (with popout play window).

Hyperlinks

Create links to other doc pages or to external pages.

HTML Inserts

Insert HTML for 'catchall' situations.

Deployment

You have real freedom to deploy in very cool ways using Doctasm.

Desktop Web App

Your docs will run on any desktop browser.

Mobile Web App

If you access from a mobile device, it will automatically display in a different way, and one that works beautifully in small screens.

Deploy to App Stores

How cool is this? With a click of a button, you can turn your Doc into an App Store ready App that runs on iOS, Androidn and Windows Phone. Think of the opportunities!

Deploy to Doctasm or Private Server

You are free to deploy your Docs to the Doctasm platform or download them and run on your own Windows, Mac or Linux servers.

Search

All page titles are ready to search and you can add tags to each page for targeted searching.

Staging & Production Modes

You can specify a Staging server (on Doctasm or your own server) for testing your Docs before pushing them to into Production.

Advanced Features

Here are some other advanced features in the Doctasm goodie bag.

  • Headers & Footers

    You can include fully customizable Headers and Footers for your Docs.

  • Roll your own styling

    As we use standard web technologies, you can create any styling you want by editing our CSS templates.

  • Breadcrumbs

    You can switch on a breadcrumbs bar to show your current opsition within the Table of Contents.

  • Page Maximizing

    The user can press a button to toggle the Table of Contents and page content to fill the entire screen.

  • Next & Previous Pages

    You can ask Doctasm to display special links to take you to the next and previous pages.
