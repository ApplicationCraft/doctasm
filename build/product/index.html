Deployment

You have real freedom to deploy in very cool ways using Doctasm.

Desktop Web App

Your docs will run on any desktop browser.

Mobile Web App

Docs will adapt beautifully and automatically on smartphones.

Deploy to App Stores

With a click of a button, you can turn your Doc into an App Store ready App for iOS, Android and Windows Phone.

Deploy to Doctasm or Private Server

You are free to deploy your Docs to the Doctasm platform or download them and run on your own Windows, Mac or Linux servers.

Staging & Production Modes

You can specify a Staging server (on Doctasm or your own server) for testing your Docs before pushing them to into Production.
